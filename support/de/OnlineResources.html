Produkte und Lizenzierung für SQL Anywhere

SQL Anywhere ist in verschiedenen Basispaketen mit zusätzlichen Deployment-Lizenzen und separat lizenzierten Komponenten erhältlich. Wählen Sie zu Beginn das Basispaket aus, das Ihren Anforderungen am besten entspricht. Basispakete sind in zwei Lizenzierungsmodellen erhältlich: (a) Server- und Seats-Modell und (b) Pro-Prozessor-Lizenz.

Das Server- und Seats-Modell umfasst einen SQL Anywhere-Netzwerkserver und eine angegebene Anzahl von Benutzerlizenzen. Jeder dieser Benutzer wird als Network Seat lizenziert, um den Zugriff des Clients auf den SQL Anywhere-Server zu ermöglichen. Außerdem wird jeder dieser Benutzer für SQL Anywhere Personal Server with Synchronization und UltraLite with Synchronization lizenziert.

Alternativ können Kunden eine Pro-Prozessor-Lizenz des SQL Anywhere-Servers erwerben. Mit diesem Paket wird der SQL Anywhere-Netzwerkserver auf einem System für die Anzahl der erworbenen Pro-Prozessor-Lizenzen lizenziert. Die Anzahl der Benutzer, die sich als Networked Seats mit dem Server verbinden können, ist unbegrenzt, sodass sich das Paket ideal für größere Client/Server- und Web-basierte Deployments eignet oder für Kunden, die eine Pro-Prozessor-Lizenzierung bevorzugen, um nicht Benutzer oder Seats zählen zu müssen.

SQL Anywhere-Basispakete enthalten darüber hinaus Lizenzen für eine Reihe von Design- und Verwaltungstools wie Sybase Central und PowerDesigner Physical Data Model.

Deployment-Lizenzen

Lizenzen für weitere Benutzer oder für zusätzliche Komponenten, wie z.B. zusätzliche Network Seats (Clients), Netzwerkserver, Personal Server, Synchronisation und UltraLite, können nachträglich erworben werden.

Support für Entwickler

Online-Support
SAP bietet einen reaktionsschnellen und zeitnahen technischen Support an, um Sie bei der raschen und kostensparenden Entwicklung und Einführung von Weltklasselösungen zu unterstützen. Diese Unterstützung wird in mehreren Stufen angeboten, die sich nach Ihren geschäftlichen Anforderungen richten. Standardangebote für Jahresverträge können Produktaktualisierungen, mehrere Kontaktstellen beim Kunden und eine unbegrenzte Anzahl von Fällen (Cases) umfassen, während Premium-Angebote darüber hinaus erweiterte Stunden, raschere Antwortzeiten und direkten Experten-Support beinhalten.

SQL Anywhere Tech Corner: Developer Community
Der Tech Corner bietet einen zentralen Kontaktpunkt für technische Informationen, sodass Sie schneller an die benötigten Informationen kommen und umfassende Unterstützung bei Entwicklung und Deployment von Lösungen für Server, PC, mobile Geräte und fernverwaltete Büroanwendungen finden. Experten aus der ganzen Welt nehmen an der Developer Community teil und bilden so die zentrale Informationsquelle für die Entwicklung von Anwendungen und Frontline-Umgebungen. Die Community bietet nicht nur Zugriff auf die neuesten EBFs, Betasoftware, Evaluierungssoftware, Dienstprogramme und vieles mehr, sie ist auch ein Diskussionsforum, in dem zu zahlreichen Themen über Newsgroups, Webcasts und Veranstaltungen Erfahrungen ausgetauscht werden.

SQL Anywhere-Forum
Das Forum wird von SQL Anywhere Customer Service & Support überwacht und ist eine nützliche Ressource, um Entwicklungsfragen mit der Entwickler-Community zu diskutieren.

DocCommentXchange
Über DocCommentXChange können Sie sich SQL Anywhere-Dokumentation ansehen und Kommentare einfügen sowie Kommentare von anderen Benutzern lesen.

SQL Anywhere-Produktinformationen
Hier finden Sie die neuesten Informationen über SQL Anywhere mit Datenblättern, Whitepaper-Veröffentlichungen und Hinweisen zu kostenlosen Downloads, aktuellen Veranstaltungen, Messen, Anwenderberichten, Pressemitteilungen usw.

SQL Anywhere Partner-Programm
Das SQL Anywhere Partner-Programm bietet seinen Mitgliedern Mehrwertdienste und Programme, die auf branchenführender Technologie basieren. Das SQL Anywhere Partner-Programm umfasst Vertriebs- und Marketingprogramme sowie Dienste, um die Marktposition der Allianzmitglieder zu verbessern. SAP hilft damit den Mitgliedern des Partner-Programms, neue Chancen besser zu nutzen und gemeinsam erfolgreicher zu agieren.

SAP Professional Services
Mit SAP Professional Services können Sie während der Design-, Entwicklungs- und Deploymentphase Ihrer SQL Anywhere-Projekte maßgeschneiderte Dienste in Anspruch nehmen:

  • Planung und Durchführung eines Upgrades auf die neueste Version von SQL Anywhere
  • Bereitstellung von Wissenstransfer zum Verständnis der SQL Anywhere-Architektur
  • Sicherstellung von Leistung und Qualität der Umgebung
  • Unterstützung bei Design und Entwicklung Ihrer MobiLink-Synchronisationslösung
  • Konfiguration und Implementierung der Hochverfügbarkeit mit SQL Anywhere

Schulung und Zertifizierung
SAP bietet SQL Anywhere-Schulungen an, um Sie bei der Entwicklung und beim Deployment Ihrer Anwendungen zu unterstützen. Je nach Niveau der Teilnehmer können Standardkurse oder spezielle Schulungsprogramme angeboten werden, die von Datenbankfunktionen über Synchronisation bis zum Deployment auf Handhelds reichen.
