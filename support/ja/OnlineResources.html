SQL Anywhere の製品とライセンス

SQL Anywhere は、いくつかの基本パッケージと、追加の展開ライセンスおよび別途ライセンスされるコンポーネントという形で提供されます。まず、お客様の要件に最もよく合う基本パッケージを選択します。基本パッケージには、(a) サーバ／シート、(b) プロセッサベースの 2 種類のライセンスモデルがあります。

サーバ／シートモデルの場合、SQL Anywhere ネットワークサーバとそれぞれの数のユーザライセンスが含まれています。各ユーザはネットワークシートとしてライセンスされ、SQL Anywhere サーバにクライアントアクセスできます。また各ユーザには、同期および同期付き Ultra Light を含む SQL Anywhere パーソナルサーバもライセンスされます。

代わりに、SQL Anywhere サーバ CPU ライセンスを選ぶこともできます。このパッケージでは、1 台のマシン上の SQL Anywhere ネットワークサーバが、購入した CPU ライセンス数だけライセンスされます。サーバにネットワークシートとして接続できるユーザの数に制限はなく、より大規模なクライアント／サーバ型の展開や Web ベースの展開、またはユーザやシートを数えるよりプロセッサベースのライセンスが好ましい場合に最適です。

SQL Anywhere の基本パッケージには、Sybase Central、PowerDesigner 物理データモデルなど、いくつかの設計および管理ツールのライセンスも含まれています。

展開ライセンス

ユーザ数を増やしたりコンポーネントを追加したりする必要がある場合でも、問題ありません。ネットワークシート (クライアント)、ネットワークサーバ、パーソナルサーバ、同期、Ultra Light には、追加ライセンスのご用意があります。

SQL Anywhere パートナープログラム
