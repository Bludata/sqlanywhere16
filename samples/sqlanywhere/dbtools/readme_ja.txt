			     データベースツールのサンプル

目的
----
このディレクトリには、データベースツールライブラリの呼び出しおよび
コンパイル方法を示すためのデータベースツールアプリケーションが含ま
れています。このアプリケーションは、SQL Anywhere のサンプルデータベー
スのバックアップを C:\Temp ディレクトリ (Windows の場合) または /tmp 
ディレクトリ (UNIX の場合) に作成します。

アプリケーションのソースコードは main.cpp にあります。

必要条件
--------
Windows の場合、このサンプルは Visual Studio .NET 2003 または 
Visual Studio 2005 がインストール済みであることと、環境変数 VCINSTALLDIR
が設定済みであることを前提としています。これらは、Visual Studio のイン
ストール時に実行されますが、Visual Studio によってインストールされた
vcvars32.bat ファイルを使用して実行することもできます。

ソースコードは、他のコンパイラでもコンパイルできます。

UNIX 専用のビルド手順については、build.sh ファイルを参照してください。

処理手順
--------
build.bat (Windows の場合) または build.sh (UNIX の場合) を実行すると、
アプリケーションをコンパイルします。

run.bat (Windows の場合) または run.sh (UNIX の場合) を実行すると、ア
プリケーションを実行します。データベースのバックアップファイルが 
C:\TEMP に作成されます。

clean.bat (Windows の場合) または clean.sh (UNIX の場合) を実行すると、
生成されたすべてのファイルを削除します。
