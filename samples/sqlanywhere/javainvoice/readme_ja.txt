		       データベースでの Java の使用のサンプル

目的
----
このフォルダーには、「データベースでの Java の使用」で使用するサンプ
ルのソースコードが含まれています。

必要条件
------------
このサンプルは、以下を前提としています。
 - 環境変数 SQLANY16 に SA フォルダーが設定されている。
   これは、SQL Anywhere のインストール時に設定されます。
 - "SQL Anywhere 16 Demo" データソースが定義されている。
   これは、SQL Anywhere のインストール時に設定されます。
 - Java コンパイラー (javac) を含む JDK がインストール済みである。
 - JAVA_HOME 環境変数に JDK インストールフォルダーが設定されている。
 - JAVAC 環境変数に JDK の javac 実行形式が設定されている。
 - demo データベース起動前に、CLASSPATH環境変数にこのフォルダーが設
   定されている。

処理手順
--------
UNIX 上でこのサンプルを実行する場合、以下の手順の .bat を .sh に置き換
えてください。

build.bat を実行すると、Java クラス・ファイルをコンパイルします。それ
をデータベースにインストールし、このクラスファイル内の Java メソッド
用に cover 関数を作成します。

run.bat を実行すると、Java メソッドを呼び出します。

clean.bat を実行すると、レポートファイル以外の生成されたすべてのファ
イルを削除します。
